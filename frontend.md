# Frontend

Этот документ описывает архитектуру и стек фронтенд-приложений проекта.

## Обзор

Директория `frontend/` представляет собой монорепозиторий, содержащий два независимых React-приложения:

1.  **`admin-panel/`**: Панель администратора.
2.  **`merchant-panel/`**: Кабинет мерчанта.

Оба приложения построены на едином технологическом стеке и используют общие подходы к разработке, но имеют разное назначение и целевую аудиторию.

---

## 1. Панель администратора (`admin-panel`)

### Назначение

**Панель администратора** — это комплексный внутренний инструмент для управления всей платежной системой. Он предоставляет сотрудникам доступ к следующим функциям:

-   Управление мерчантами, пользователями и их ролями.
-   Управление платежными провайдерами и их группами.
-   Мониторинг и управление финансовыми транзакциями и платежами.
-   Обработка диспутов и просмотр логов.
-   Настройка системных параметров, таких как курсы валют.
-   Просмотр общей дашборд-аналитики по системе.

### Технический стек

-   **Язык**: TypeScript
-   **Фреймворк**: React 19
-   **Сборщик**: Vite
-   **Роутинг**: `react-router-dom`
-   **Управление состоянием**:
    -   **Серверное состояние (Server State)**: `TanStack Query` (для кэширования, инвалидации и фонового обновления данных).
    -   **Клиентское состояние (Client State)**: `Zustand` (для управления глобальным состоянием UI, аутентификации).
-   **HTTP-клиент**: `axios`
-   **UI-компоненты**:
    -   **Стилизация**: Tailwind CSS.
    -   **Библиотека**: Компоненты построены по методологии, схожей с **Shadcn/UI**, с использованием `tailwind-merge`, `clsx` и `class-variance-authority`.
    -   **Иконки**: `lucide-react`.
-   **Таблицы**: `TanStack Table` (для сложных, сортируемых и фильтруемых таблиц).
-   **Интернационализация (i18n)**: `i18next` и `react-i18next` с поддержкой русского и английского языков.
-   **Уведомления**: `sonner` (тост-уведомления).
-   **Drag-and-Drop**: `@dnd-kit`.

## 2. Кабинет мерчанта (`merchant-panel`)

### Назначение

**Кабинет мерчанта** — это клиентское приложение для мерчантов, подключенных к платформе. Оно позволяет им:

-   Просматривать дашборд с ключевыми метриками.
-   Отслеживать свои платежи и балансовые операции.
-   Управлять кошельками и методами оплаты.
-   Выполнять операции Payout, Payin и вывода средств.
-   Формировать отчеты.
-   Настраивать свой профиль и управлять пользователями своей команды.

### Технический стек

Базовый стек аналогичен admin-panel (React 19, TypeScript, Vite, Tailwind CSS, TanStack Query, Zustand, axios, i18next, sonner, react-router-dom v7), но без ряда компонентов админ-панели:

- Нет `@tanstack/react-table` (сложные таблицы не нужны мерчанту)
- Нет `@dnd-kit` (drag-and-drop не нужен)
- Нет `i18next-browser-languagedetector` (определение языка браузера)

## Запуск и развертывание

### Локальный запуск для разработки

Для запуска любого из приложений необходимо перейти в его директорию, установить зависимости и запустить dev-сервер:

```bash
# Для запуска панели администратора
cd frontend/admin-panel
npm install
npm run dev

# Для запуска кабинета мерчанта
cd frontend/merchant-panel
npm install
npm run dev
```

### Развертывание (Deployment)

Проект использует **Docker** для контейнеризации. В директории `frontend/docker/` находятся `Dockerfile` и конфигурации Nginx для каждого приложения. Сборка и запуск Docker-контейнеров является предпочтительным способом развертывания в production-среде.
