# Rate Service — Сервис курсов валют

> **Стек:** Python
> **Репо:** `rate/`
> **Последнее обновление:** 2026-04-15 (автосинк)

---

## 1. Обзор

Rate Service — микросервис для получения и предоставления курсов валют. Используется агрегатором для расчёта курсов обмена в транзакциях.

---

## 2. Источники курсов (`source_exchanges`)

Формат поля `source_exchanges` в конфигурации провайдеров:

| Формат | Описание |
|--------|----------|
| `<exchange>` | Обычный источник (напр. `binance`, `kaspi`) |
| `static-<number>` | Фиксированный курс, возвращает число напрямую |
| `binancekaspi130-<side>` | Binance/Kaspi со стороной сделки, 1.30x множитель. `<side>` = `buy` или `sell` |

### Формат `binancekaspi130-<side>` (добавлен 2026-04-14)
Позволяет задать сторону напрямую в строке источника:
```
binancekaspi130-buy   → source=binancekaspi130, side=buy
binancekaspi130-sell  → source=binancekaspi130, side=sell
```
Это позволяет не передавать `side` отдельным параметром.
